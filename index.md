## About My Project
We build systems that learn an individual's decision-making and progressively act on their behalf. The core is a domain-agnostic engine: it observes a user's choices and the reasons they give, induces explicit decision policies (condition → action → rationale), and graduates each policy along an autonomy ladder — proposing, then confirming, then acting independently — while demoting any policy the user overrides. When it encounters a situation that is novel or contradicts an existing policy, it flags it for human review rather than silently generalizing from a single event. Learned policies persist and transfer across domains, so the same system adapts to a new coordination problem by swapping a lightweight domain adapter rather than being rebuilt. I evaluate it in LLM multi-agent simulations of everyday coordination, where simulated participants have private motives, misreport, and renege — conditions under which naive automation fails.
